### 싱글턴: 객체의 인스턴스가 오직 하나만 생성되도록 보장

인스턴스는 무조건 하나다. (복사 금지)

싱글턴을 만드는 방식으로는 세가지가 있다.

1. private static

public으로 열어두지만 생성자를 호출하도록 하지는 못하게 한다.

```java
// President.java
public class President {
    // 1. 유일한 인스턴스를 미리 만들어 공개(public)합니다.
    public static final President INSTANCE = new President();

    // 2. 다른 사람이 새로 선출하지 못하게 생성자를 막습니다(private).
    private President() {
        System.out.println("대통령이 선출되었습니다.");
    }

    public void sayHello() {
        System.out.println("안녕하세요, 대통령입니다.");
    }
}

// Main.java
public class Main {
    public static void main(String[] args) {
        // President.INSTANCE를 통해 항상 같은 객체를 사용합니다.
        President p1 = President.INSTANCE;
        President p2 = President.INSTANCE;

        p1.sayHello(); // "안녕하세요, 대통령입니다."
        p2.sayHello(); // "안녕하세요, 대통령입니다."

        System.out.println(p1 == p2); // true (p1과 p2는 완전히 같은 객체입니다)
    }
}
```

2. 정적 팩터리 메서드

getInstance만 public으로 열어서 이 함수로만 접근할 수 있도록 한다.
```java
// President.java
public class President {
    // 1. 유일한 인스턴스를 숨겨둡니다(private).
    private static final President INSTANCE = new President();

    // 2. 생성자도 막습니다(private).
    private President() {}

    // 3. "반장 나와주세요!"라고 외치는 공식적인 방법을 만듭니다.
    public static President getInstance() {
        return INSTANCE;
    }

    public void sayHello() {
        System.out.println("안녕하세요, 대통령입니다.");
    }
}

// Main.java
public class Main {
    public static void main(String[] args) {
        // getInstance()를 통해 항상 같은 객체를 받습니다.
        President p1 = President.getInstance();
        President p2 = President.getInstance();

        System.out.println(p1 == p2); // true
    }
}
```
3. 열거형

자바 문법 자체를 이용해서 유일한 객체를 만든다.
```java
// President.java
public enum President {
    INSTANCE; // 'INSTANCE'가 유일한 객체의 이름입니다.

    public void sayHello() {
        System.out.println("안녕하세요, 대통령입니다.");
    }
}

// Main.java
public class Main {
    public static void main(String[] args) {
        President p1 = President.INSTANCE;
        President p2 = President.INSTANCE;

        p1.sayHello();
        System.out.println(p1 == p2); // true
    }
}
```
