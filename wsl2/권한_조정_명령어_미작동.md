# chmod,chown 미작동 에러 Fix

```
sudo umount -l /mnt/c
sudo umount -t drvfs C: /mnt/c -o metadata
```

설정 변경 후 터미널 재시작

💫 drvfs 는 WSL과 Windows 파일 시스템 간의 상호 운용성을 지원하도록 만든 파일 시스템 플러그인
